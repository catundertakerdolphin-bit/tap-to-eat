# TAP TO EAT

Соревновательное мукбанг-шоу: тапай по столу-сетке, держи комбо, избегай тухлятины, обгоняй ботов. Между раундами качаешь тело в магазине.

**Играть онлайн:** https://catundertakerdolphin-bit.github.io/tap-to-eat/

## Локально
- Просто открыть `merged/index.html` двойным кликом (file://) — играет, дебаг-сейв уходит в localStorage + скачивание.
- Или с авто-записью конфига в проект: `node merged/server.js` → http://localhost:4321 (есть `POST /save-config` → пишет `merged/config.json` + бэкап в `config-backups/`).

## Внутри
- Стол = сетка слотов; еда занимает футпринт из слотов, многослойная еда (лобстер, лазанья) снимается по слоям.
- 3 твари: 🐙 Ктулху (AoE-укус крестом), 🦪 Дети фермы (рой ×1.5), 🦄 Пони (комбо ×2, charm).
- Раж-метр, сытость/переполнение, магазин с провизорными покупками, дебаг-панель (⚙) с живыми ползунками.
- Вся графика — эмодзи-плейсхолдеры, точки замены помечены `// ASSET`.
